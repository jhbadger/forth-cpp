info
  ( -- )
  BadgerForth -- a lightweight single-file Forth interpreter in C++.
  Architecture:
    Compiles to compact 24-byte bytecode instructions (Op enum).
    Each Ins holds: opcode, integer field, pointer, and string index.
    String data lives in a parallel side-table (Code::sv), keeping
    the hot instruction stream cache-dense.
    Dispatch uses an iterative interpreter with an explicit call stack
    (no C++ recursion), and an array-backed data and return stack.
    Hot opcodes (+, -, dup, swap, @, ! etc.) are handled inline in
    the dispatch loop without function calls.
  Memory model:
    A byte vector (heap) serves as the address space.
    Cell size is 4 bytes. BASE is at heap[0], STATE at heap[4].
    HERE and ALLOT operate on byte addresses.
  String model:
    Strings live in a separate string_table vector.
    Words operate on integer indices into this table.
    s" pushes an index; type prints by index; s+ concatenates.
  Dictionary:
    Words stored in a deque<Entry> for pointer stability.
    A name->index hash map gives O(1) lookup.
    Four entry kinds: PRIM, WORD, DEFINING, CREATE.
    Each entry has a unique integer execution token (xt).
  Double-cell:
    64-bit values represented as (lo hi) pairs on the stack.
    Full set of ANS double-cell words: d+, d-, dnegate, dabs,
    d=, d<, du<, dmax, dmin, d2*, d2/, m+, s>d, d>s,
    um*, um/mod, sm/rem, fm/mod.
  Pictured numeric output:
    <# # #s sign hold holds #> produce formatted number strings.
  Exception handling:
    catch/throw provide structured exception handling.
    abort" throws when a flag is non-zero.
  Locals:
    { a b c -- } syntax inside definitions.
    -> stores into a named local.
  Marker:
    marker saves a dictionary snapshot; executing it forgets
    everything defined after the marker was created.
  Control flow:
    if/else/then, begin/until, begin/while/repeat, begin/again,
    do/loop/+loop, ?do, leave, exit, recurse,
    case/of/endof/endcase.

quit
  ( -- )
  Throws a "quit" exception, returning control to the REPL.
  Useful for exiting nested include files or evaluation.

throw-str
  ( idx -- )
  Internal helper used by ABORT". Throws using the string at idx.
  Equivalent to: -1 throw  with idx already on the stack.
  Not normally called directly.
